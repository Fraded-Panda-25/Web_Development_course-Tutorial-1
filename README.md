# Simple HTML Starter

This repository contains a minimal HTML boilerplate file (`index.html`) that can be used as a starting point for any web project.

## 📄 File Structure

- **index.html**  
  A basic HTML5 document setup with:
  - `<!DOCTYPE html>` declaration
  - `<html>` element with language attribute
  - `<head>` section including:
    - UTF-8 character encoding
    - Responsive viewport meta tag
    - Title placeholder
  - Empty `<body>` section ready for content
